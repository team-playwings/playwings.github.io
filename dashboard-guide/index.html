로그인

https://dashboard.playwings.co.kr/로 접속하여, 사전에 접근권한 신청한 계정으로 로그인 진행


Wingsorder > Analytics

  1. 메뉴에서는 귀사에서 공급한 상품의 판매 현황 및 결과를 확인할 수 있습니다.(판매시작일시, 판매종료일시, 상품명, 옵션별 구매 및 취소수, 총결제수, 총 결제취소수, 실 결제수, 총 결제 금액, 총 환불 금액, 실 매출금액)
  2. 우측 상단의 캘린더 버튼으로 기간 설정하여 확인 가능하며, Export Chart, Export Table 버튼으로 차트와 테이블을 내보내기 할 수 있습니다.

Wingsorder > Vouchers

  1. Vouchers 테이블에는, 귀사에서 진행한 모든 상품의 결제 내역 확인 및 예약 관리가 가능합니다.
  2. 테이블에서는 구매번호, 구매일시, 구매자, 구매상태, 컨펌번호, 체크인&체크아웃, 취소일시, 상품명, 구매옵션, 구매수, 구매합계, 제공가, 상품금액, 할인금액, 결제금액, 연락처등 확인 가능합니다.
  3. 우측상단의 내보내기를 통하여 csv 파일로 저장 가능합니다.
  4. 사용자의 구매상태로는, 6가지 상태가 존재하며 각 상태의 설명은 하기와 같습니다.

PREPARED 사용자가 결제 페이지에 머문 상태
PAID 결제 완료된 상태
CANCELED 결제 후, 취소한 상태
FAILED 네트워크 혹은 시스템문제로 결제 실패한 상태
CONFIRMED 결제완료건(PAID) -> 예약 담당자가 예약확정 완료한 상태
OVERBOOKED 사용자 투숙일 지정일의 재고 부족 혹은 호텔 내부 사정으로 오버부킹 처리된 상태

Wingsorder > Vouchers : 예약확정

  1. 첫번째 열의 구매번호를 클릭하면 예약 상세 페이지로 이동할 수 있습니다.
  2. 컨펌번호의 경우, 예약담당자가 예약확정 마지막 단계에 직접 입력하므로 공란으로 노출 됩니다.
  1. 예약 상세페이지로 진입하여, 사용자의 구매정보, 구매자정보, 예약정보 등을 최종 확인 하고 우측 상단의 예약확정 버튼을 클릭합니다.
  2. 예약확정 버튼 클릭 시, 사용자에게 안내 될 알림톡을 미리 확인할 수 있습니다. (사용자 수신 알림톡 이미지 참고)
  3. 컨펌번호 입력란은 필수값으로, 호텔 내부적으로 확인할 확정번호를 직접 입력해주신 후에 우측 하단의 예약확정 버튼을 클릭하면 예약확정이 완료됩니다. (PAID → CONFIRMED)

Wingsorder > Vouchers : 오버부킹

(플레이윙즈 관리자만 사용 가능)

  1. 사용자가 지정한 투숙일에 재고가 부족하거나 호텔 내부 상황으로 해당일에 투숙이 불가한 경우, 오버부킹 처리를 통한 사용자에게 투숙일 재지정을 요청할 수 있습니다. (사용자 수신 알림톡 이미지 참고)
  2. 현재 배포 버전에서는 플레이윙즈 관리자만 사용가능한 기능으로, 오버부킹 처리 필요시 플레이윙즈 서비스매니지먼트팀으로 요청 부탁드립니다.

Wingsorder > Vouchers : 구매취소

  1. 구매취소 기간이 지난 경우, 사용자가 직접 취소 불가하므로 CS인입을 통한 예약관리자가 수동으로 취소 가능합니다.
  2. CS는 플레이윙즈 서비스매니지먼트팀에서 접수하며, 수동취소가 필요한 명단을 일별로 업데이트하여 전달드립니다. 명단 확인 후, 예약관리자가 수동으로 취소처리 필요합니다.
  3. 구매 취소 버튼 클릭 시, 사용자에게 안내 될 알림톡을 미리 확인할 수 있습니다. (사용자 수신 알림톡 이미지 참고)
  4. 취소 완료 시, 상태값은 “CANCELED”로 변경됩니다.

Wingsorder > Vouchers : 예약재확정

  1. CONFIRMED상태의 예약건의 경우, 체크인/체크아웃 일정 변경, 컨펌번호 수정 등이 필요할 시, 예약재확정이 가능합니다.
  2. 예약재확정 버튼 클릭 시, 사용자에게 안내 될 알림톡을 미리 확인할 수 있습니다. (사용자 수신 알림톡 이미지 참고)

Wingsorder > Vouchers : 메모 & 히스토리

  1. 우측 상단의 메모 기능을 통해 해당 예약건 관련하여 간단한 메모 작성이 가능합니다.
  2. 좌측 하단의 히스토리 기능을 통해 변경 및 업데이트 기록을 확인할 수 있습니다.
