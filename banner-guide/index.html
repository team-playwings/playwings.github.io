메인 최상단 배너

메인 최상단 배너는 앱 접속 시 가장 먼저 접하게 되는 빅사이즈 배너로 앱에서 가장 중요한 메인 영역에 위치합니다. 특가상품, 프로모션, 콘텐츠 등 다양한 소재로 노출됩니다.

배너 사이즈 1080 X 1320 (px)
파일 포맷 PSD
텍스트 가이드 NanumSquareRound ExtraBold 90pt (Line-Height 120pt)
1. 텍스트의 컬러는 가독성 있는 컬러를 사용합니다.
2. 샘플 PSD의 텍스트 레이어의 위치 이동 및 형태 변경을 금지합니다.
3. 텍스트의 길이는 최대 2줄이며, 배너 좌우 마진 72px을 넘어서는 안됩니다.
4. 해당 폰트 이용이 불가능 할 경우, San-serif 폰트로 제작 후 전달 주시면 추후 플레이윙즈에서 변경하여 사용됩니다.
배경 가이드 1. 눈에 피로를 유발하는 지나치게 채도가 높은 색상 사용은 불가능합니다.
2. 배경 이미지가 텍스트의 가독성을 해치지 않아야 합니다.
3. 배경 이미지에 텍스트 혹은 로고가 포함되지 않아야 합니다.
4. 텍스트 가독성을 위해 배경에 Dim처리가 가능합니다.
로고 가이드 1. PSD에 포함된 로고만 활성화하여 사용해야합니다.
2. 위치, 크기 변경은 불가하며 컬러는 가독성 있게 변경 가능합니다.

안드로이드 종료 배너

2. 배경 이미지가 텍스트의 가독성을 해치지 않아야 합니다.
3. 배경 이미지에 텍스트 혹은 로고가 포함되지 않아야 합니다.
4. 텍스트 가독성을 위해 배경에 Dim처리가 가능합니다.
로고 가이드 1. PSD에 포함된 로고만 활성화하여 사용해야합니다.
